# Propriedade associativa da soma de convolução
Código para verificar a propriedade associativa da soma de convolução

O código pega 3 vetores (aleatórios ou não, cê pode escolher) e aplica a convolução em ordens distintas, mostrando o resultado no final. Se os resultados forem iguais, a propriedade associativa foi verificada.

## Executando
basta executar

```python3 associativa.py```

certifique-se de que o python esteja instalado
